Jquery selector là gì

Css Selector là bộ chọn css để chọn các thẻ html mà các bạn mong muốn ví dụ :.class,#id,tag_name(body,div,.._

Jquery selector là bộ chọn của jquery để chọn các thẻ html mà các bạn mong muốn

    css=>jquery
    .class=>$(.class)
    #id=>$(#id)
      tag_name ( div ) => $("tag_name") $("div")
    .classA .classB => $(".classA .classB")
    .classA > .classB => $(".classA > .classB")
    chú ý là tất cả select tên class , tên id , tên thẻ đều phải nằm "" hoặc ''
nhưng mà document và window thì không cần trong "" hay ''
    $(document).ready(); là gì
    $() là cú pháp của selector query document là đại diện của toàn trang html
    ready có nghĩa là khi mà trang web được tải xong và ta sẽ viết các lệnh jquery hoặc js trong method này để đảm bảo js được chạy sau khi web được tải xuống
