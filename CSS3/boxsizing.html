box-sizing có giá trị măc định là conten-box

conten-box=> thuộc tính width trong css là độ rộng của content

content-box => thuộc tính height trong css là độ cao của content

border-box => thuộc tính width trong css là tổng độ rộng của box model

border-box=> thuộc tính height trong css là tổng độ cao của boxmodel
